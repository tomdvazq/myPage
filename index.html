First of all, thank you for visiting my site 🙏

My name is Tomás, I'm 23 years old and I'm from Argentina. Since I finished high school I went through many places to get here. The IT world has always been close to me, but for a long time I resisted getting to know it in depth.

But always, sooner or later, one arrives at the port where one must arrive.

How did I learn about programming? 💻

My first approach was at the age of nine when I set up a personal version of Habbo Hotel, a multiplayer game that had released its code so that players could create their own versions.

Without really knowing what I was doing I started using PHP and SQL to be able to host users within my version of the game. Not knowing so much about it, it was obviously hacked at the time. 😂

Studies 📚

I started studying in a self-taught way via YouTube with various tutorials that I found on the platform, thanks to this I got to know freeCodeCamp and took the HTML and CSS course. It was in March 2022 that I decided to get serious and started studying at CoderHouse academy, getting certified as a web developer. Currently I continue to take courses at CoderHouse and have created a self-taught habit of learning.

Bootstrap

Like many, Bootstrap was the first framework I used in CSS. Personally I don't like it because it messes up the readability of the HTML code. Beyond my preferences, I usually use it if I need something punctual and fast.

This is not to say that I reject Bootstrap completely, I've worked with it quite a bit, but it doesn't occupy a place in my top preferences.

Tailwind CSS

Unlike Bootstrap, Tailwind seems to me to be a much neater and cleaner framework with respect to code translation.

In addition to the above highlights, I believe that this project created by Wathan and Schoger offers much more extensive customization.

Sass

Truly my favorite tool! Sass, just like the name says, is awesome.

In itself, the fact of being able to work the project in such an organized way thanks to imports and nestings is a real madness. In addition, being able to add variables, mixins, make extends... SCSS is my best friend when it comes to styles.

Pure CSS

Before learning about and using the above tools I devoted a great deal of attention to understanding the roots of cascading style sheets.

When it comes to layout I excel in the use of grid, flex and finally box-modeling for small details.
